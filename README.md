## workspace automater is a command line tool for creating workspaces assembling multiple apps to run with a single command

# Usage

## Setup : copy and past the following command in your terminal

git clone https://github.com/abdelalimah/workspaceAutomater.git && sudo bash workspaceAutomater/automater.sh setup && cd && rm -rf workspaceAutomater

## Creating a workspace
automater create mySpace phpStorm HazeOver ...

## Launching a workspace
automater start mySpace

## Closing a workspace(on upcoming release)
automater close work_space_name

## Deleting a workspace(on upcoming release)
automater delete work_space_name
##
to edit or delete a workspace please refer to /.automater_workspaces on your root directory and modify it with your favorite text editor
